## 🔄 End-to-End R&D Pipeline

I am fully equipped to drive complex projects through every stage of development:
1. **Requirements & Tech Specs (TRS):** Formalizing goals, hardware system architecture, and risk analysis.
2. **Hardware Engineering:** Designing electrical schematics, routing multi-layer PCBs, and preparing manufacturing data (Gerber, BOM).
3. **Mechanical Design:** Building precise enclosure models in Fusion 360/FreeCAD tightly integrated with custom board shapes.
4. **Firmware Engineering:** Developing robust C/C++ bare-metal or RTOS code, wireless connectivity protocols, and low-power sleep state management.
5. **Fullstack Web:** Creating backend engines to capture telemetry via MQTT and designing responsive React operational dashboards.
6. **Lab Debugging:** Full hardware test-bench verification using professional diagnostic tools (Digital Oscilloscope, Logic Analyzer).
